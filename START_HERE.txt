STEP 1: START REDIS
──────────────────

Option A - Windows:
  > redis-server

Option B - WSL2:
  > wsl
  $ redis-server

Option C - Docker:
  > docker run -d -p 6379:6379 redis:latest

Keep this running in background!


STEP 2: START DJANGO SERVER
───────────────────────────

In command prompt at d:\company\chats\:

  > daphne -b 0.0.0.0 -p 8000 config.asgi:application

You should see:
  Listening on TCP address 127.0.0.1:8000


STEP 3: OPEN BROWSER
────────────────────

Go to:  http://localhost:8000

That's it! 🎉

Problem: "WebSocket connection failed"
Solution: Make sure Redis is running on port 6379

Problem: "Port 8000 already in use"
Solution: Use different port:
  > daphne -b 0.0.0.0 -p 8001 config.asgi:application

Problem: "Module not found"
Solution: Reinstall dependencies:
  > pip install -r requirements.txt

Problem: "Cannot create room - name already exists"
Solution: Room names are unique. Choose a different name.
